Comandos de Git

Comando Descripción Ejemplo
git init Crea un nuevo repositorio Git vacío. git init
git add Agrega cambios al área de preparación. git add archivo.txt
git commit Registra los cambios en el repositorio. git commit-m "Mensaje de confirmación"
git push Envía los cambios locales al repositorio remoto. git push origin master
git pull Obtiene los cambios remotos y los fusiona con los cambios locales. git pull origin master

Creación de claves SSH

Para crear un par de claves SSH, sigue los siguientes pasos:

  1. Abre una terminal en tu equipo.
  2. Ejecuta el siguiente comando para generar una nueva clave SSH:
    3. ssh-keygen -t rsa -b 4096 -C "tu_correo_electrónico"
  3. Sigue las instrucciones para guardar la clave en un archivo.

Comandos de Docker

Para instalar Docker, sigue los siguientes pasos:

  1. Descarga e instala Docker para tu sistema operativo desde el sitio web oficial de Docker.
  2. Asegúrate de que Docker se esté ejecutando correctamente ejecutando el comando:
    3. sudo docker run hello-world

A continuación, se muestran algunos de los comandos más comunes de Docker:

Comando Descripción Ejemplo
docker build Construye una imagen Docker a partir de un archivo Dockerfile. docker build -t myimage:1.0 .
docker run Ejecuta un contenedor Docker basado en una imagen. docker run -p 8080:80 myimage:1.0
docker ps Muestra todos los contenedores Docker en ejecución. docker ps
docker stop Detiene un contenedor Docker en ejecución. docker stop container_id
docker rm Elimina un contenedor Docker existente. docker rm container_id
